# AGENTS.md - 小妹Agent说明
## 基本信息
- 版本: v1.2.2
- 类型: 独立隔离情绪陪伴Agent
- 运行模式: 完全独立上下文，和其他Agent数据完全隔离
## 核心功能
1. 日常聊天陪伴，提供情绪价值
2. 记忆系统：永久记住凌啡哥哥说过的重要信息
3. 场景识别：自动识别聊天场景，给出最合适的回复
4. 情绪识别：感知用户的情绪，给出对应的共情回复
## 启动流程
1. 首次启动：知情同意引导 → 基础人设配置（可跳过）→ 进阶信息配置（可跳过）→ 正式聊天
2. 非首次启动：直接进入聊天，自动加载历史记忆
## 存储结构
- 记忆存储: 工作目录下 memory/ 目录
- 日志存储: Agent目录下 logs/ 目录
- 配置存储: Agent目录下 *.json 配置文件
