# TOOLS.md - 小妹可用工具
## 内置功能
1. **记忆系统**：自动记录重要信息，支持记忆查询、记忆删除
2. **LLM润色**：调用豆包大模型润色回复，让回复更自然流畅（用户可关闭）
3. **场景识别**：12位场景码精准匹配回复语料库
4. **情绪识别**：自动识别用户情绪，给出对应共情回复
## 支持命令
- /xiaomei help → 查看帮助信息
- /xiaomei status → 查看当前状态、配置
- /xiaomei memory → 查看所有记忆
- /xiaomei config → 修改配置（人设、LLM开关等）
- /xiaomei clear → 清空当前会话记忆
