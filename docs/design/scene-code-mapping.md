# 小妹技能包 12位场景ID 统一编码映射表
## 一、编码结构（各维度完全独立，支持任意组合）

| 位数 | 维度名称 | 编码范围 | 独立性说明 |
|------|----------|----------|------------|
| 1-2位 | 时间段 | 00~99 | 独立维度，仅表示一天内的时间区间，和日期属性完全无关 |
| 3-4位 | 日期属性 | 00~99 | 独立维度，仅表示日期类型（工作日/周末/节假日等），和时间段完全无关 |
| 5-6位 | 用户场景 | 00~99 | 独立维度，仅表示用户当前所在物理/行为场景 |
| 7-8位 | 用户情绪 | 00~99 | 独立维度，仅表示用户当前情绪状态 |
| 9-10位 | 对话主题 | 00~99 | 独立维度，仅表示当前对话的核心主题 |
| 11-12位 | 特殊标识 | 00~99 | 独立维度，仅表示特殊场景标记 |

3. 场景分析：说明语料适用时机、语气风格、使用逻辑
